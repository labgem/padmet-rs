# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## v0.1.0 

### Added

- Parsing a `.padmet` file with padmet-rs
- Get nodes of a certain kind (compound, reaction, pathway)
- Get relations from a node, or to a node, and relation of a certain kind (e.g, "is_a")
- Get the set of reactions of a pathway
- Get relations from or to a node
